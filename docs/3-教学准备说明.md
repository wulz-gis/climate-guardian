# 教学准备统一说明

> 本文档集中说明所有课次的教学准备要求。为保持主文档清晰，详细内容从 `docs/2-课程详细内容.md` 中移出，统一在此进行维护与更新。

## 目标与范围
- 统一规范每课的“导入视频、素材数据、交互图表模板、电子记录”准备要求。
- 提供目录结构、命名与版权合规建议，便于教师快速检索与替换。
- 适用于全课程（第1–第25课）。后续新增或调整在本文件更新。

## 统一规范（建议）
- 导入方式：每课“情景导入（5分钟）”统一使用短视频（1–2分钟）。
- 素材类型：图片（高清）、示例数据（CSV/JSON）、交互模板（PPT/HTML/Notebook）、电子记录表（表格/表单）。
- 命名规则：`lesson-XX-intro.mp4`、`lesson-XX-sample-data.csv`、`lesson-XX-charts-template.pptx`、`lesson-XX-record.xlsx`。
- 最低质量要求：视频清晰≥720p，字幕可读，音量适中；数据字段含列名与单位；模板含最少示例页。

## 目录结构建议
```
assets/
  videos/
    lesson-01-intro.mp4
    lesson-02-intro.mp4
    ...
  images/
    lesson-XX-*.png
  data/
    lesson-XX-sample.csv
  templates/
    charts/
      lesson-XX-template.pptx
    forms/
      lesson-XX-record.xlsx
```
- 按课次分组，命名统一，避免同名覆盖。
- 建议在 `README.md` 维护资源清单与来源链接。

## 版权与合规
- 使用外部素材需注明来源与授权范围（教学/非商业）。
- 建议保存离线副本，避免课堂网络不佳导致播放失败。
- 若素材含敏感信息（个人数据/地理位置信息），需脱敏或获取许可。

## 视频资源建议（通用）
- 题材：与课题直观关联，突出核心概念或现象（如：风的形成、降雨雷达、Keeling曲线、冰川前沿对比、海冰范围变化、北极熊栖息地压力）。
- 时长：1–2分钟，信息密度适中，结尾含1–2个引导问题。
- 形态：动画科普或实景剪辑，优先含字幕与简洁图示。

## 数据与交互模板（通用）
- 数据格式：CSV/JSON，包含列名、单位、时间戳/地理字段（如适用）。
- 交互模板：
  - 图表：折线、柱状、堆叠/面积、热力图、地图。
  - 操作：筛选、时间轴拖动、图例高亮、事件标注、计算面板（如能耗→CO₂）。
- 校验清单：字段合法、单位一致、图例清晰、交互不依赖外网接口（或提供离线方案）。

## 按单元准备要点（简要）
- 第1–2单元（基础认知与观测）：
  - 视频：气温/气候要素概览、观测方法示意。
  - 数据：温度/降水/风向风速示例表。
  - 模板：折线、风玫瑰、柱状图。
- 第3单元（灾害识别）：
  - 视频：高温/热浪、洪水/干旱典型场景。
  - 数据：温湿度、降雨强度、SPI简化指标。
  - 模板：风险分级面板、地图示意。
- 第4–6单元（温度韵律与长期变化）：
  - 视频：季节变化、长期增温趋势与事件影响。
  - 数据：小时/日/季/年气温序列与异常值。
  - 模板：多尺度折线、滑动平均、事件标注。
- 第7–8单元（温室气体与关系）：
  - 视频：排放结构、能源与排放、Keeling曲线与温度异常。
  - 数据：部门排放、能耗与系数、CO₂浓度与温度异常。
  - 模板：堆叠柱/树图、计算面板、相关分析。
- 第9单元（冰川与海冰）：
  - 视频：冰川前沿对比、海冰范围时序。
  - 数据：冰川长度/质量平衡、海冰面积/多年冰比例。
  - 模板：时间轴、地图热力图、前后对比。
- 第10单元（生态影响）：
  - 视频：物候变化、北极熊栖息地压力与行为变化。
  - 数据：物候记录、海冰指标与栖息地分布。
  - 模板：时间轴、地图叠加、图文卡片。

## 维护与更新
- 当某课教学准备变更时，仅更新本文件对应资源清单与链接，不在 `2-课程详细内容.md` 中重复。
- 建议每次变更后运行一致性自检（资源命名、目录存在、链接有效）。

## 数据来源与下载渠道（精简版）

- Our World in Data（OWID）：https://ourworldindata.org/ — 能源、排放等主题开放数据，配套 CSV 与 GitHub 镜像。
- NOAA：https://www.noaa.gov/ — GML（CO₂ 趋势）、NCEI（气候档案）、NSIDC（海冰与雪盖）。
- NASA / Earthdata：https://earthdata.nasa.gov/ — GISTEMP、卫星产品（GES DISC）、SVS 可视化素材。
- Meteostat：https://meteostat.net/ — 气象观测时序与统计；另可用 Copernicus CDS 获取再分析数据。
- Zenodo：https://zenodo.org/ — 开放科学数据托管与 DOI。
- Figshare：https://figshare.com/ — 研究数据与图表托管，注意许可（如 CC BY）。
- GitHub：https://github.com/ — 代码与数据整理；建议固定到 release/commit 并保留离线副本。
- ESSD（期刊）：https://www.earth-system-science-data.net/ — 数据论文与描述，原始数据常托管于机构库。
- Nature Scientific Data：https://www.nature.com/sdata/ — 数据论文与开放数据链接。
- Natural Earth：https://www.naturalearthdata.com/ — 全球矢量/栅格地理底图用于教学地图。
- WorldClim：https://worldclim.org/ — 历史与气候平均栅格数据（温度、降水等）。

### 使用与合规（简要）
- 课前检查外链可用性（HTTP≥200，非零字节），优先离线副本。
- 标注来源与权利方，核对许可范围（教学/非商业）。
- 原始数据放置 `data/data/`，处理后放置 `assets/data/`；命名遵循 `lesson-XX-*`。

如需我生成各课的具体资源清单（含链接与占位文件），请告知数据源或授权范围，我可以批量补充到 `assets/` 并在本文件登记。
