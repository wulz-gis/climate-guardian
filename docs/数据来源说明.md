# 真实数据来源说明（教学资源）

本文档记录本项目中使用的真实数据来源、下载链接与处理方法，确保教学资源可溯源、可复现。所有数据均保存离线副本：原始数据在 `data/` 目录，处理后的教学用资源在 `assets/` 目录。

更新时间：2025-11-01（UTC）

## NASA GISTEMP 全球温度异常（Land–Ocean: Global Means）

- 官方页面：https://data.giss.nasa.gov/gistemp/
- 直接下载（CSV，全球陆海表面温度异常，含年均 J-D）：
  - https://data.giss.nasa.gov/gistemp/tabledata_v4/GLB.Ts+dSST.csv
- 本地存放：`data/data/gistemp_glb_ts_dsst.csv`
- 教学资产：
  - 第12课 CSV（含滑动均值）：`assets/data/lesson-12-sample.csv`
  - 第21课 CSV（与 CO₂ 合并）：`assets/data/lesson-21-sample.csv`
  - 第15课图像（全球温度异常折线）：`assets/images/lesson-15-evidence.png`
- 处理说明：
  - 解析表头 `Year, Jan...Dec, J-D, D-N, DJF, MAM, JJA, SON`，取 `J-D` 为年度温度异常（°C）。
  - 对第12课，基于年度异常计算 5 年简单滑动均值。
  - 合并第21课时与 CO₂ 年均数据按年份对齐。
  - 字体配置：macOS 使用 `Heiti TC`，`axes.unicode_minus=False`。

## NOAA GML Mauna Loa CO₂（月度与年度均值）

- 项目页面（趋势与下载）：https://gml.noaa.gov/ccgg/trends/
- 月度 CSV（列：`year,month,decimal date,average,deseasonalized,ndays,sdev,unc`）：
  - 说明页面含多个下载入口（CDO/GML）；本项目使用 GML 提供的月度文件。
- 本地存放：`data/data/noaa_mauna_loa_co2_monthly.csv`
- 教学资产：
  - 第21课 CSV（与温度异常合并）：`assets/data/lesson-21-sample.csv`
  - 第21课图像（双轴：CO₂ 与温度异常）：`assets/images/lesson-21-co2-temp.png`
- 处理说明：
  - 跳过注释行（以 `#` 开头），对同一年份的 `average` 求算术平均作为年度 CO₂（ppm）。
  - 与 GISTEMP 年度异常按年份合并，生成教学用 CSV 与双轴图。

## 数据来源总览（精简）

- Our World in Data（OWID）：能源与排放等主题开放数据（含 CSV 与 GitHub 镜像）。
- NOAA：GML（CO₂ 趋势）、NCEI（气候档案）、NSIDC（海冰与雪盖）等。
- NASA / Earthdata：GISTEMP、各类卫星产品（GES DISC）、SVS 可视化素材。
- Meteostat：气象观测时序与统计；另可用 Copernicus CDS 获取再分析数据。
- Zenodo：开放科学数据托管与 DOI。
- Figshare：研究数据与图表托管，注意许可（如 CC BY）。
- GitHub：研究代码与数据整理；建议固定到 release/commit 并留离线副本。
- ESSD（期刊）：数据论文与描述，原始数据通常托管于机构库（如 PANGAEA、Zenodo）。
- Nature Scientific Data：数据论文与开放数据链接。
- Natural Earth：全球矢量/栅格地理底图，用于教学地图与叠加。
- WorldClim：历史与气候平均栅格数据（温度、降水等）。

## 地理与气候背景数据（地图/栅格）

- Natural Earth（https://www.naturalearthdata.com/）
  - 内容：行政界线（国家/省）、海岸线、河流、地形底图等矢量与栅格。
  - 用途：教学地图底图、叠加区域指标（温度异常、海冰范围等）。
  - 许可：公共领域（Public Domain），可自由使用；仍建议标注来源以示尊重。

- WorldClim（https://worldclim.org/）
  - 内容：历史气候与气候平均（v2/v2.1），含温度与降水等变量，多分辨率网格。
  - 用途：区域气候背景地图、栅格统计与示例分析（如年均温度阶梯图）。
  - 许可：遵循项目说明与引用规范；下载后建议保留 README 与元数据。

## 海平面（GMSL）来源与下载详情（已纳入）

- 数据集：Global Mean Sea Level Trend from Integrated Multi-Mission Ocean Altimeters TOPEX/Poseidon, Jason-1, OSTM/Jason-2, Jason-3, and Sentinel-6 Version 5.2
  - 短名：`MERGED_TP_J1_OSTM_OST_GMSL_ASCII_V52`
  - DOI：`10.5067/GMSLM-TJ152`
  - Landing Page：https://doi.org/10.5067/GMSLM-TJ152
  - 数据文件（受保护）：https://archive.podaac.earthdata.nasa.gov/podaac-ops-cumulus-protected/MERGED_TP_J1_OSTM_OST_GMSL_ASCII_V52/GMSL_TPJAOS_5.2.txt
- 本地原始文件：`data/data/nasa_gmsl_ascii.txt`（离线副本，文本/ASCII）
  - 源文件名：`GMSL_TPJAOS_5.2.txt`
  - 下载日期：2025-11-01（UTC）
  - SHA256：`c19af03af18d8d2a12e29c3ecef9c1c4faa48ef7efb52860205949ae57f2d805`
- 处理与输出：
  - 脚本入口：`python3 scripts/process_data_assets.py`
  - 解析策略：跳过 HDR 头部，识别小数年份列与 GIA 应用的 60 天平滑列，聚合为年度平均（单位自适应为 mm）。
  - 产出文件：`assets/data/lesson-15-sample.csv`（列：`年份, 温度异常/°C, 海平面/mm`）
  - 配套图像：`assets/images/lesson-15-evidence.png`

备注：也兼容 `NASA_SSH_GMSL_INDICATOR`（DOI: `10.5067/NSIND-GMSV1`），若采用该备选数据源，脚本会自动将 cm 转换为 mm。
